# ❤️ FLAMES Love Game

A beautiful, interactive **FLAMES love calculator** built with pure HTML, CSS, and JavaScript. Enter two names and discover your relationship — Love, Friend, Affection, Marriage, Enemy, or Sister!

## ✨ Features

- 💘 **FLAMES Algorithm** — animated letter-by-letter elimination with heart tiles
- 🌐 **Multilingual** — English & Tamil support
- 🎵 **Sound Effects** — themed songs for each result
- 🎨 **Dynamic Themes** — background changes based on your result
- 🎆 **Fireworks & Animations** — skyshot rockets, floating hearts, result overlays
- 💗 **Heart Cursor** — custom animated heart cursor with trail effects
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- 🔗 **Shareable Links** — share your result with a unique URL
- 🔇 **Mute Button** — toggle sound on/off anytime
- ⚠️ **Smart Alerts** — centered blur-backdrop alerts with auto-dismiss

## 🛠️ Built With

- HTML5
- CSS3 (animations, glassmorphism, backdrop-filter)
- Vanilla JavaScript + jQuery
- Web Audio API
- Canvas API

## 🚀 Getting Started

1. Clone the repository
```bash
   git clone https://github.com/your-username/flames-game.git
```
2. Add your song files inside the `songs/` folder:
```
   songs/en/   → English songs
   songs/tam/  → Tamil songs
   songs/hi/   → Hindi songs
```
3. Open `index.html` in your browser — no server needed!
